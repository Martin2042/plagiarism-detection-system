# 🔍 Plagiarism Detection System

## 📌 Overview
This project is a web-based plagiarism detection system developed using Python and Django. It allows users to upload text, documents, or URLs and analyzes content similarity using text comparison algorithms.

---

## 🚀 Features
- Detect plagiarism in text and documents  
- Upload files or provide URLs  
- Fast similarity comparison  
- User-friendly interface  

---

## 🛠 Tech Stack
- Python  
- Django  
- JavaScript  
- HTML & CSS  
- SQLite  

---

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/Martin2042/plagiarism-detection-system.git

2. Navigate to the project:
   cd plagiarism-detection-system

3. Install dependencies:
   pip install -r requirements.txt

4. Run the server:
   python manage.py runserver
